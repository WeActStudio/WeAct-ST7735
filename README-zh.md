## 引脚定义描述

|引脚|描述|
| :--:|:--:|
|1.GND| 电源地|
|2.VCC| 电源正 支持 3.3V 到 5V|
|3.SCL| SPI 时钟输入|
|4.SDA| SPI 数据输入|
|5.RST| 复位|
|6.DC| 数据/命令选择|
|7.CS| SPI 片选输入|
|8.BL| 背光控制输入|
